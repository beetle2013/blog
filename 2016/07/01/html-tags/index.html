html标签
html标签对于web开发人员来说，再熟悉不过了，但全部掌握的估计不多。HTML说它简单也不简单，它是网页的基础，是骨架。HTML要尽量语义化，标签如果正好表明它内容的含义，对阅读他的人或程序（通常是搜索引擎）会非常有利。同时各个浏览器会对html标签赋予行为，使用时要注意。

!– — comment 注释标签用于在源文档中插入注释。注释内容不会被浏览器显示。为代码编写注释的好处是，有助于增强代码的可读性。
!DOCTYPE !DOCTYPE 文档声明必须位于 HTML5 文档中的第一行，也就是位于 html 标签之前。该标签告知浏览器文档所使用的 HTML 规范。
a Anchor,a 标签定义超链接，用于从一个页面链接到另一个页面。a 标签最重要的属性是 href 属性，它指定链接的目标 url 。
abbr Abbreviation,abbr 标签最初是在 HTML 4.0 中引入的，表示它所包含的文本是一个更长的单词或短语的缩写形式。
address address 标签定义文档作者或拥有者的联系信息。通常的做法是将 address 元素添加到网页的头部或底部。
area area 元素始终嵌套在 map 标签内部。area 标签定义图像映射内部的区域（图像映射指的是带有可点击区域的图像）。
article article 标签是 HTML 5 的新标签, article 标签的内容独立于文档的其余部分。
aside aside 标签定义其所处内容之外的内容。aside 标签的内容应该与附近的内容相关。
audio audio 标签是 HTML 5 的新标签, audio 标签想网页中插入声音，比如音乐或其他音频流。
b Bold,b 标签定义粗体的文本。b 标签用于强调某些文本。如果您希望把文本标记为相比其他文本更为重要，应该使用 strong 标签。
base base 标签为页面上的所有链接规定默认地址或默认目标。base 标签必须位于 head 元素内部。
bdi Bi-directional Isolation,bdi 标签是 HTML5 中的新标签。bdi 标签允许您设置一段文本，使其脱离其父元素的文本方向设置。
bdo Bi-Directional Override,bdo 标签覆盖默认的文本方向。通过设置 dir 属性来设置文本的显示方向。
blockquote blockquote 标签之间的所有文本都会从常规文本中分离出来，经常会在左、右两边进行缩进，而且有时会使用斜体。
body body 标签定义文档的主体。 body 标签包含文档的所有内容，比如文本、超链接、图像、表格、列表等。在 HTML 5 中，删除了所有 的特殊属性。
br Break,br 标签插入简单的换行符。br 标签是一个空标签，意味着它没有结束标签。br 标签常用来输出空行
button button 标签定义按钮。您可以在 button 标签内放置内容，比如文本或图像。但是在 form 中通常使用 input 标签创建按钮。
canvas canvas 标签定义图形，比如图表和其他图像。canvas 标签只是图形容器，您必须使用脚本来绘制图形。
caption caption 标签定义表格的标题。caption 标签必须直接放置到 table 标签之后。通常标题会居中显示在表格上方。
cite Citation,cite 标签定义作品（比如书籍、歌曲、电影、电视节目、绘画、雕塑等等）的标题，通常斜体显示。
code code 标签常用来格式化代码文本。
col Column,col 标签为表格中的一个或多个列定义属性值。通过使用 col 标签，可以向整个列应用样式，而不需要重复为每个单元格或每一行设置样式。
colgroup Column Group,colgroup 标签，可以向整个列应用样式，而不需要重复为每个单元格或每一行设置样式。只能在 table 元素内使用 colgroup 标签。
datalist datalist 标签定义选项列表。请与 input 元素配合使用该元素，来定义 input 可能的值。
dd Definition Description,dd 标签定义一个定义列表中对项目的描述。
del Delete,del 标签定义文档中已删除的文本。与 ins 标签一起使用，描述文档中的更新和修正。
dfn Definition,dfn 标签可标记那些对特殊术语或短语的定义。
div Division,div 标签常用于组合块级元素，以便通过样式表来对这些元素进行格式化。通常使用 div + css 对网页进行布局。
dl Definition List,dl 标签定义一个定义列表。dl 标签内部使用 dd 和 dt 定义列表中的项目。
dt Definition Term,dt 标签定义一个定义列表中的一个项目。dd 标签描述这些项目。
em Emphasis ,em 呈现为被强调的文本。
embed embed 标签是 HTML 5 中的新标签。embed 标签定义嵌入的内容，比如插件。
fieldset fieldset 标签会在相关表单元素周围绘制边框。
figcaption Figure Caption,figcaption 标签是 HTML 5 中的新标签。figcaption 标签定义 figure 元素的标题（caption）。
figure figure 标签是 HTML 5 中的新标签。figure 标签规定独立的流内容（图像、图表、照片、代码等等）。
footer footer 标签是 HTML 5 中的新标签。footer 标签定义 section 或 document 的页脚。
form form 标签标签用于创建供用户输入的 HTML 表单。form 标签包含多个表单元素，如：button,input,keygen,object,output,select,textarea。
h1,h2,h3,h4,h5,h6 Headings,h1,h2,h3,h4,h5,h6 标签用来定义文档中的标题。h1 定义最大的标题。h6 定义最小的标题。
head head 标签是所有头部元素的容器。位于 head 内部的元素可以包含 title,base,link,meta,script,style等。
header header 标签是 HTML 5 中的新标签。header 标签定义文档的页眉（介绍信息）。
hr Horizontal Rule,hr 标签定义内容中的主题变化，并显示为一条水平线。在 HTML 4.01 中，hr 标签仅仅显示为一条水平线。
html Hypertext Markup Language,html 标签告知浏览器这是一个 HTML 文档。html 元素是 HTML 文档中最外层的根元素。
i Italic,i 标签定义的文本在 html 文档中呈现为斜体，以便与文档中其余部分区别开来。
iframe Inline Frame,iframe 标签创建包含另一个文档的行内框架。在 HTML 5 中，仅仅支持 src 属性。
img Image,img 标签定义 HTML 页面中的图像。
input input 标签规定 form 表单中用户可输入数据的输入字段。根据不同的 type 属性，输入字段可以是文本字段、复选框、密码字段、单选按钮、按钮等。
ins Insert,ins 标签定义文档的其余部分之外的插入文本。通常能够与 del 标签一起使用，来描述对文档的更新和修正。
kbd Keyboard Input,kbd 标签定义键盘文本。它表示文本是从键盘上键入的。它经常用在与计算机相关的文档或手册中。
label label 标签为 input 元素定义文字标识，label 标签的 for 属性应该等于相关元素的 id 元素，以便将它们捆绑起来。
legend legend 元素为以下元素定义标题（caption）：fieldset、figure、details 。 legend 元素为以下元素定义标题（caption）：fieldset、figure>、details。
li List Item,li 标签定义列表项，有序列表 ol 和无序列表 ul 中都使用 li 标签。 HTML 5 中，li 标签不再支持 type 属性。
link link 标签定义文档与外部资源之间的关系。link 标签大多数时候都用来连接 css 样式表。link 标签只能位于head 标签中。
main main 标签规定文档的主要内容。
map map 标签用于定义图像中可点击的热点区域。map 标签的 name 属性与 img 标签的 usemap 属性相关联，以创建图像与映射之间的关系
mark mark 标签是 HTML 5 中的新标签。mark 标签定义带有记号的文本。请在需要突出显示文本时使用 mark 标签。
meta Metadata,meta 标签位于文档的头部 head 标签中，meta 标签的属性用来定义有关页面的元信息，比如 字符集，关键词，描述，重定向等。
meter meter 标签是 HTML 5 中的新标签。meter 标签定义度量的范围，既可以在元素的文本中，也可以在 min/max 属性中定义。
nav Navigation,nav 标签是 HTML 5 中的新标签。nav 标签定义导航链接的部分。
noscript noscript 元素用来定义在脚本未被执行时的替代内容（文本）。此标签用于可识别 script 标签但无法支持其中脚本的浏览器。
object object 定义一个嵌入的对象。请使用此元素向您的 HTML5 页面添加多媒体。
ol Ordered List,ol 标签定义有序列表。在 HTML 5 中，不再支持 compact 和 type 这两个属性
optgroup Options Group,optgroup 标签定义选项组。此元素允许您组合选项。当您使用一个长的选项列表时，对相关的选项进行组合会使处理更加容易。
option option 标签定义下拉列表中的一个选项。在 HTML 5 中，option 标签也用于新元素 datalist 中。
output output 标签是 HTML 5 中的新标签。output 标签定义不同类型的输出，比如脚本的输出。
p Paragraph,p 标签定义段落。在 HTML 5 中不再支持 p 标签的 align 属性。
param Parameter,param 标签允许您为插入 XHTML 文档的对象规定 run-time 设置，也就是说，此标签可为包含它的 object 标签提供参数。
pre Preformatted Text,pre 标签可定义预格式化的文本。pre 标签的一个常见应用就是用来表示计算机的源代码。
progress progress 标签是 HTML 5 中的新标签。progress 标签定义运行中的进度（进程）。
q Quotation,q 标签定义一个短的引用。浏览器经常会在这种引用的周围插入引号。
samp Sample,samp 标签定义样本文本。
script script 标签用于定义客户端脚本，比如 JavaScript。script 元素既可包含脚本语句，也可以通过 src 属性指向外部脚本文件。
section section 标签定义文档中的节（section、区段）。比如章节、页眉、页脚或文档中的其他部分。
select select 标签创建下拉列表。请在 form 元素中使用此标签来接受用户的输入。
small small 元素定义旁注信息，并显示为更小的文本。对于由 em 元素强调过的或由 strong 元素标记为重要的文本，small 元素不会取消对文本的强调，也不会降低这些文本的重要性。
source source 标签是 HTML 5 中的新标签。source 标签为媒介元素（比如 video 和 audio）定义媒介资源。
span span 标签用于对文档中的行内元素进行组合,以便通过样式表 css 对它们进行格式化。
strong strong 标签定义重要的文本。
style style 标签定义 HTML 文档的样式信息。scoped 属性是 HTML 5 中的新属性，它允许我们为文档的指定部分定义样式，而不是整个文档。
sub Subscript,sub 标签可定义下标文本。
sup Superscript,sup 标签可定义上标文本。
table table 标签定义 HTML 表格。一个简单的 HTML 表格包括 table 元素，一个或多个 tr、th 以及 td 元素。
tbody Table Body,tbody 标签，可以将表格分为一个单独的部分。tbody 标签可将表格中的一行或几行合成一组。在 HTML 5 中，不支持 tbody 标签的任何属性。
td td 标签定义 HTML 表格中的标准单元格。thead 元素中只能使用 th 标签，不允许使用 td 标签。
template template 标签可用于HTML模板。
textarea textarea 定义一个多行的文本输入区域。文本区中的默认字体是等宽字体 (fixed pitch)。
tfoot Table Footer,tfoot 定义表格的页脚（脚注），tfoot 内部必须拥有 tr 标签！tfoot 表亲必须位于 table 标签内。
th Table Header Cell,th 标签定义 HTML 表格中的表头单元格。th 元素中的文本呈现为粗体并且居中。th 标签在 tr 标签中定义。
thead Table Header,thead 标签在 table 标签中使用，定义表格的表头。thead 内部必须拥有 tr 标签！
time time 标签是 HTML 5 中的新标签。time 标签定义公历的时间（24 小时制）或日期，时间和时区偏移是可选的。
title title 位于 head 部分，用来定义文档的标题。title 元素在所有 HTML 文档中是必需且唯一的。
tr Table Row,tr 标签定义表格中的行，一个 tr 元素包含一个或多个 td 或 th 标签。
track track 标签是 HTML 5 中的新标签。track 标签为诸如 video 元素之类的媒介规定外部字幕文件或其他包含的文版文件。
ul Unordered List,ul 标签定义无序列表。请使用 CSS 来定义列表的显示样式。
var Variable,var 标签可用于定义变量。
video video 标签是 HTML 5 的新标签。video 标签定义视频，比如电影片段或其他视频流。
wbr Word Break Opportunity,wbr 标签是 HTML 5 中的新标签。wbr 规定在文本中的何处适合添加换行符。
